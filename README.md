# label_check

#### 介绍
学习项目：用pyqt5编写UI界面,识别条形码，并校验条码是否正确；识别文件中的条码和USB扫描枪的条码。

#### 软件架构
pyqt5 ui文件构建界面,部分widget使用qss样式,检验条码规则可以进行使用自定义.


#### 安装教程

1. git clone  git@gitee.com:hl_lance/label_check.git
2. cd label_check
3. python -m pip install pyqt5 

#### 使用说明

1.  python main.py  xxxx
